  • Jet Set Radio Practice Tool (Source not available)

    • Role: Developer

    This is a DLL that can be injected into the game "Jet Set Radio HD". It hooks into the game's Direct3D9 instance to display info on screen and allows the user to press buttons that modify the values of certain memory addresses to give the player certain abilities (no time limit).

  • Linux Auto Splitter Timer (Source available here)

    • Role: Co-Developer

    This is a timer program for Linux that allows the user to load Lua scripts. These Lua scripts contain information such as a video game's addresses and logic to control the timer. Using these Lua files, the program knows when to start the timer, pause the timer, unpause the timer and "split".
